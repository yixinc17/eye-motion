## 交互

- **4:3 框内**：移动鼠标 = 人脸中心 (cx, cy)
- **出框**：触发丢失逻辑（眼看最后位置 → 停 1s → 缓慢回正）
- **右侧面板**：可调参数（PRD 清单）

## 设计文档

详见 **[DESIGN.md](./DESIGN.md)**：眼-体协同机制、分区设计、流程图。

## 管线概要

输入 → 归一化 → 去抖 → 输入平滑(EMA) → 分区 → 目标生成 → 输出平滑 → 绘制

## 分区概要

- 死区：中心小方块 \|nx\|,\|ny\| < 0.12
- 眼区：0.12 ≤ \|nx\| < zone_head_min
- 头区：\|nx\| ≥ zone_head_min（眼先动 → 延迟 → 头转 → 眼归位）

## 可视化

- 眼睛：240×240，瞳孔偏移 ±30px，仅 eyeball + pupil（参考 eye-demo）
- 头部：3D rotateY，-30° ~ +30°
